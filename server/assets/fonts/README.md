# Bundled fonts

Bundled so the server can render label PNGs without system fonts (the production
Docker image ships none). Both are licensed under the SIL Open Font License 1.1.

- `IBMPlexSans.ttf` — IBM Plex Sans (variable), (c) IBM Corp.
- `Spectral-SemiBold.ttf` — Spectral, (c) Production Type.
